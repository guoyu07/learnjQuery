  1. div1与div2垂直相邻，div1的margin-bottom=10px,div2的margin-top=8px 则div1与div2的上下距离为10px，而不是18px；水平方向上同理(margin与margin取最大)
  2. body包含div1，body的padding-top=8px,div1的margin-top=10px 则div1距离body的距离为18px;(padding与margin取二者之和)
  3. 设置padding不会影响内容的width与height,但会影响实际的页面占地大笑
  4. offset()方法是绝对定位，position()方法是相对定位，如果所有父类都是默认定位(非relative,absolute)，则position()方法等同于offset()方法
  5. position()方法在计算距离的时候会减去margin
